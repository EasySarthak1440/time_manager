# Time Manager

A comprehensive, modern web-based time management application built with React 19, TypeScript, and Tailwind CSS.

## üöÄ Features

-   **Clock:** A real-time digital clock to keep you on schedule.
-   **Alarm:** Set and manage multiple alarms with audio notifications.
-   **Timer:** Highly configurable countdown timer for various tasks.
-   **Stopwatch:** Precision stopwatch with lap timing capabilities.
-   **Focus Mode:** Pomodoro-style focus sessions to boost productivity.
-   **Settings:** Global settings to customize your experience, including theme and notification sounds.
-   **Responsive Design:** Fully responsive UI that works across desktop and mobile devices.

## üì¶ Getting Started

### Prerequisites

-   Node.js (v18 or higher recommended)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/time-manager.git
    cd time-manager
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:5173](http://localhost:5173) in your browser.
